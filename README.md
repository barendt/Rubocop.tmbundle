Rubocop.tmbundle

Supports
--------
* severities

Install
-------

set $TM_RUBY to your ruby directory

    gem install rubocop
    mkdir -p ~/Library/Application\ Support/Avian/Bundles
    cd ~/Library/Application\ Support/Avian/Bundles
    git clone git://github.com/fazibear/Rubocop.tmbundle.git
